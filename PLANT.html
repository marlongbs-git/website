Dynamic Solar Shading System with Maximum Power Point Tracking

Team PLANT's objective was to develop a proof-of-concept for an automated solar panel shading system that adjusts panel tilt to protect greenhouse plants. Coupled with a maximum power point tracking (MPPT) system, it ensures maximum energy extraction from the panels.

As part of the team, I led the electronics design, developing a DC-DC converter with integrated sensing and control capabilities to support dynamic MPPT algorithm execution

PCB Design

PCB design and bring-up using Altium Designer. Manufactured all boards with hand soldering.

Schematic Capture

Schematic design and simulation using Altium Designer and LTSpice for circuit analysis and testing using intergrated circuits. Use of PLECS for power electronics simulation.

Hardware Debugging

Hardware validation using oscilloscopes, DMMs, and signal generators for PCB debugging and signal integrity testing.

Key Achievements

  • Designed schematic and layout using Altium Designer and LTSpice
  • Implemented a DC-DC synchronous buck converter with integrated voltage and current sensing
  • Designed and hand-soldered 8+ breakout boards for component validation
  • Iteratively revised PCB layout across multiple bring-up cycles to improve testability
  • Achieved active voltage and current sensing within +3.3V specifications
  • Firmware-ready architecture supporting dynamic MPPT control and algorithm implementation

This capstone project focused on designing a DC-DC converter to accept variable input voltages from organic solar panels supplied by the ASCA Group. With no prior PCB design experience, this project was a deliberate deep dive into electrical hardware — from circuit theory to a fully routed board.

The initial design used a two-switch buck-boost topology with independent gate drivers. Component sizing for the capacitors, inductor, MOSFETs, and high-side gate drive circuitry were all hand-calculated. Altium Designer was self-taught through online resources, and the layout was refined iteratively with guidance from power electronics advisors. The first board failed during bring-up, likely due to solder bridging on fine-pitch components during hand assembly. A second population attempt yielded the same result, with the high-side driver (10-MSOP package) suspected as the root cause — whether from routing or soldering. Given the cost of the components and the difficulty of diagnosing a fried IC, a full redesign was the right call. The redesigned board switched to a synchronous buck topology using a half-bridge driver, simplifying the gate drive circuit and reducing cost. The new layout prioritized testability — adding oscilloscope test points, easy ground references, and 0-ohm resistor bridges to allow staged bring-up of individual subsections before full power-on.
